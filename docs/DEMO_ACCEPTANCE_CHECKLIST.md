# ClipMind 演示验收清单

逐页验证系统是否就绪。打开 Web（`http://<bind-addr>:<WEB_PORT>`），按顺序核对。

## 核心链路（端到端）
- [ ] **素材库** `/assets`：能看到 NAS 索引的真实视频；对一个素材触发拆镜头。
- [ ] **镜头库** `/shots`：拆出的镜头有真实关键帧/缩略图；点开能播放代理视频。
- [ ] AI 打标：镜头有 AI 描述/产品/场景/动作/风险标签（需已配置真实 AI）；可人工审核确认/修改。
- [ ] **产品库** `/products`：产品存在并能与素材/镜头关联。

## 搜索 / 收藏
- [ ] **智能搜索** `/search`：自然语言搜索返回相关镜头，匹配度合理（语义检索生效，非纯词法降级）。
- [ ] **保存搜索**：保存当前搜索条件 → 在列表里看到 → 载入后表单条件被正确恢复 → 重新运行。
- [ ] **收藏** `/favorites`：从镜头/搜索结果收藏 → 在收藏页按类型筛选看到 → 移除收藏后镜头/素材仍在。

## 脚本匹配 / 导出
- [ ] **脚本匹配** `/script`：贴入脚本 → 拆段 → 每段有候选镜头 → 选择/锁定 → 重新匹配时锁定不被覆盖。
- [ ] 缺口段：无合适镜头的段落显示缺口原因与补拍建议（不编造素材）。
- [ ] **多格式导出**：剪辑清单导出 **CSV / Excel(XLSX) / JSON / Markdown / 可打印页面** 均能生成并下载；
      打开后中文正常、时间码/镜头 ID 真实。
- [ ] **打包 ZIP**：多选镜头 → 创建 ZIP → 下载 → 解压可见 `clips/`、`manifest.json`、剪辑清单；片段可播放。

## 项目 / 集合
- [ ] **项目** `/projects`：建项目，加入素材/镜头/产品/脚本。
- [ ] **静态集合**：手工挑选镜头组成集合；删除集合不删镜头。
- [ ] **动态集合**：按搜索条件创建 → 打开时实时返回当前匹配的镜头（改变素材/索引后结果随之更新）。
- [ ] **归档/恢复**：项目归档后只读，恢复后可继续编辑。

## 导出中心 / 生命周期
- [ ] **导出中心** `/exports`：集中显示片段/脚本清单/打包三类导出；可筛选、下载。
- [ ] 失败的导出可**重试**；已完成/失败的导出可**删除**（删除只删导出文件，源素材/镜头/脚本仍在）。

## 稳定性
- [ ] **重启持久化**：`scripts/nas/restart.sh` 后，项目/集合/收藏/导出/匹配选择与锁定均保留。
- [ ] **备份/恢复**：`backup.sh` 生成备份；`restore.sh` 可恢复到一致状态。
