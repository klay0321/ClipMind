# ClipMind 快速上手（业务负责人）

## 这是什么
ClipMind 把公司 NAS 里**难以检索、难以理解的原始视频素材**，变成**可搜索、可匹配、可下载**的镜头资产：
自动把长视频拆成镜头、生成预览、用 AI 给每个镜头打标签，然后你可以用自然语言搜索、按脚本匹配镜头、
导出剪辑清单与片段打包。**它不生成视频**（不做文生视频/数字人/换脸），只整理和检索你已有的真实素材。

## 怎么访问
由 NAS 管理员按《部署指南》安装后，用浏览器打开管理员给你的地址（形如 `http://<内网IP>:3000`）。
> 目前**没有登录账号体系**，请仅在公司内网/VPN 内使用，不要把地址发到公网。

## 各页面能做什么
- **素材库** `/assets`：看 NAS 里被索引的视频，触发拆镜头与分析。
- **镜头库** `/shots`：浏览拆出的镜头，看关键帧、预览代理视频、人工审核 AI 标签。
- **产品库** `/products`：维护产品，关联到素材/镜头。
- **智能搜索** `/search`：用自然语言或条件搜镜头；可**保存搜索**，以后一键重跑。
- **脚本匹配** `/script`：贴入脚本→自动拆段→为每段匹配镜头→人工选择/锁定→生成**剪辑清单**，
  可导出 **CSV / Excel / JSON / Markdown / 可打印页面**。
- **项目** `/projects`：把素材/镜头/产品/脚本/集合组织成一个项目；项目里可建**静态集合**（手工挑选的镜头）
  和**动态集合**（按搜索条件实时更新的镜头集合）。
- **导出中心** `/exports`：集中查看所有导出（片段、剪辑清单、打包 ZIP），下载、重试失败、删除记录。
- **收藏** `/favorites`：收藏素材、镜头、搜索结果或脚本匹配结果，随时回看。

## 典型用法
1. 在**智能搜索**里描述你要的画面（如"户外使用产品的特写"）→ 预览满意的镜头 → 收藏或加入项目。
2. 在**脚本匹配**里贴入脚本 → 逐段确认镜头 → 导出剪辑清单（给剪辑同事）或把片段打包成 ZIP 下载。
3. 把一次活动的素材、镜头、清单都放进一个**项目**，方便协作与复用。

## 注意
- 系统**只读**你的原始素材，绝不会修改/删除源视频。删除导出只删导出文件，不影响素材。
- 还没有"谁在用/谁下载"的用户记录（无账号体系）；这部分在后续版本规划中。
- 安装、升级、备份由 NAS 管理员用交付包中的脚本完成（见《NAS 部署指南》）。
